GuidGen-v2

Globally Unique Identifier (GUID) Generator.

What can:
1) Generate piece
2) Generate a given quantity
3) Delete piece
4) Delete everything
5) Save the entire list to the exchange buffer

Что может:

1) Формирует разово
2) Создание заданного количества
3) Удалить разово
4) Удалить все
5) Сохранить весь список в буфере обмена
